本节根据当前 `OUTPUT/logs` 与 `OUTPUT/*_random.csv` 中已经完成的实验结果整理。除特殊说明外，主结果采用 5-fold cross-validation × 5 random seeds，共 25 次独立训练的 mean ± std。所有实验统一使用层次化池化参数 n1=4、n2=2。KIBA 数据集训练耗时较长，主表中 KIBA 相关结果暂时留空，后续可在 KIBA 全部 fold 与 seed 完成后重新运行根目录脚本 `process_experiment_results.py --include-kiba` 进行补齐。分类任务（Human / C.elegans）后续也将按 n1=4、n2=2 重新运行，此处暂不填入。

## 4.1 Datasets, evaluation metrics, and implementation details

### 4.1.1 Datasets

本文在两个药物-靶标亲和力回归数据集和两个药物-靶标相互作用分类数据集上评估 HAG-DTA。Davis 与 KIBA 用于 drug-target affinity regression；Human 与 C.elegans 用于 drug-target interaction classification。

| Dataset | Task | Compounds | Proteins | Interactions | Metrics | Current status |
|---|---:|---:|---:|---:|---|---|
| Davis | Regression | 68 | 442 | 30,056 | MSE / CI / rm² | Completed |
| KIBA | Regression | 2,111 | 229 | 118,254 | MSE / CI / rm² | Pending; left blank |
| Human | Classification | 2,726 | 2,001 | 6,728 | AUROC / AUPRC / Precision / Recall | Pending rerun |
| C.elegans | Classification | 1,767 | 1,876 | 7,785 | AUROC / AUPRC / Precision / Recall | Pending rerun |

Davis 数据集包含 68 个药物分子和 442 个激酶靶点之间的结合亲和力，标签由 dissociation constant ($K_d$) 转换得到。KIBA 数据集融合多种生物实验指标，规模约为 Davis 的 4 倍，因此训练时间显著更长。Human 与 C.elegans 数据集包含正负样本，用于二分类 DTI 预测。

### 4.1.2 Evaluation metrics

对于回归任务，本文使用 MSE、CI 和 $r_m^2$：

- MSE 越低表示预测误差越小；
- CI 越高表示模型对样本亲和力排序的一致性越强；
- $r_m^2$ 越高表示外部预测拟合能力越好。

对于分类任务，本文使用 AUROC、AUPRC、Precision 和 Recall。AUROC 衡量整体二分类区分能力，AUPRC 更适合评估类别不平衡条件下的正类识别能力，Precision 与 Recall 分别反映预测准确性与召回能力。

### 4.1.3 Implementation details

当前实验使用 HAG-DTA 的 GIN 图卷积变体（`Diff_DTA_GIN`）。训练配置如下：

| Item | Regression | Classification |
|---|---:|---:|
| Random seeds | 100, 1000, 2000, 3000, 4000 | 100, 1000, 2000, 3000, 4000 |
| Folds | 5 | 5 |
| Epochs | 1000 | 1000 |
| Batch size | 512 | 256 |
| Learning rate | 0.0005 | 0.0005 |
| Optimizer | Adam | Adam |
| Early stopping patience | 50 | 50 |
| Validation interval | 5 | 5 |
| Output source | `OUTPUT/logs/davis_f*.log` | `OUTPUT/logs/human_f*.log`, `OUTPUT/logs/celegans_f*.log` |

所有实验使用论文 per-dataset 最优层次化池化参数：Davis n1=4、n2=2；KIBA n1=6、n2=2；Human 与 C.elegans n1=7、n2=3。MMD loss 权重为 β=0.05。蛋白质序列统一截断或填充至长度 1000，并输入 TextCNN 进行序列特征提取。药物分子图经过局部层次化图分支（HGNN）、深层全局图分支（DGNN）以及注意力融合模块得到最终多模态表示。分类任务后续将按 n1=7、n2=3 重新运行后再更新本节表格。

## 4.4 Ablation and MMD sensitivity analysis

当前 LOG 中可直接复查的是 MMD coefficient sensitivity。组件消融（去除 HGNN、DGNN、attention 或 protein branch）若需要写入最终论文主表，应以后续对应实验日志为准；本节暂不虚构未在当前 LOG 中完整出现的数据。

MMD loss 的权重 $\beta$ 在 Davis 上的单组敏感性实验如下：

| β | MSE ↓ | CI ↑ | rm² ↑ | Best epoch |
|---:|---:|---:|---:|---:|
| 0 | 0.2274 | 0.8906 | 0.6379 | 158 |
| 0.01 | 0.2155 | 0.8932 | 0.6733 | 401 |
| 0.05 | 0.2134 | 0.8939 | 0.6738 | 278 |
| 0.1 | 0.2236 | 0.8866 | 0.6627 | 258 |
| 0.5 | 0.2158 | 0.8925 | 0.6758 | 427 |
| 1.0 | 0.2130 | 0.8980 | 0.6819 | 700 |

从当前单组结果看，加入 MMD loss 后整体优于 $\beta=0$ 的设置，说明分布对齐约束对局部层次特征与全局图特征的融合具有积极作用。其中 $\beta=0.05$ 相比 $\beta=0$ 明显降低 MSE 并提升 CI；$\beta=1.0$ 在该组日志中取得最低 MSE 和最高 CI。由于该敏感性实验目前是单组设置，最终是否采用 $\beta=0.05$ 仍应结合多 fold 稳定性、模型设计动机以及过大 MMD 权重对主任务学习的潜在影响综合判断。

## 4.6 Complexity analysis

HAG-DTA 的计算开销主要来自以下部分：

1. 层次化 DiffPool 分支需要在 dense molecular graph 上学习 assignment matrix，并计算 link prediction loss 与 entropy regularization；
2. 全局 DGNN 分支堆叠多层图卷积以提取全局拓扑特征；
3. MMD loss 在 batch 内计算分布对齐约束；
4. 注意力融合模块需要对局部分支、全局图分支与蛋白序列分支进行动态加权。

因此，HAG-DTA 相比单分支 GNN-CNN 模型具有更高训练成本，但也能够显式建模药物分子的多尺度结构信息。KIBA 数据集由于样本数显著大于 Davis，并且当前代码中存在 KIBA 子集拆分加载逻辑，训练时间明显更长。基于当前实验进度，KIBA 的最终指标暂不纳入本节定量分析，待完整运行后再补充训练时间与性能表。

## 4.7 Visualization and interpretability analysis

当前 LOG 中保留了 Davis 数据集的训练曲线与注意力权重变化图，可用于说明模型收敛行为与多分支融合过程。

### 4.7.1 Training curves

![Davis training curves](LOG/davis_training_curves.png)

Davis 的训练曲线显示，模型在前期快速下降，并在后续 epoch 中逐步进入平稳区间。不同随机种子的曲线趋势基本一致，说明训练过程具有一定稳定性。不过，部分 fold 的最优 epoch 出现在较后阶段，说明当前训练设置仍可能存在较长的无效训练区间，后续可结合 early stopping 或更高效的验证频率策略进一步缩短训练时间。

### 4.7.2 Attention weight evolution

![Davis attention weights](LOG/davis_attention_weights.png)

注意力权重可用于观察模型在不同特征分支之间的动态分配。当前 Davis 可视化结果显示，模型在训练后期更倾向于依赖高层局部池化表示与全局图表示，而第一层细粒度局部表示和蛋白序列分支的权重相对下降。这说明 HAG-DTA 在当前任务中主要利用药物分子的高层结构片段与全局拓扑信息完成亲和力预测。

需要指出的是，注意力权重只能提供模型内部特征使用情况的间接解释，不能直接等价于真实化学因果机制。后续若用于论文中的可解释性分析，应结合具体分子案例、功能团标注或分子对接结果进一步验证。
