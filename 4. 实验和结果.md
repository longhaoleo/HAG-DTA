# 4. 实验和结果

## 4.1 Datasets, evaluation metrics, and implementation details

为全面评估所提出 HAG-DTA 模型的有效性，本文在两个药物-靶标亲和力回归数据集（Davis 与 KIBA）以及两个药物-靶标相互作用分类数据集（Human 与 C.elegans）上进行了实验验证。各数据集的任务类型、样本规模、评价指标及数据划分方式如 Table~\ref{tab:datasets} 所示。

### 数据集说明

Davis 数据集包含激酶抑制剂与蛋白靶点之间的结合亲和力信息，其亲和力标签采用 $K_d$ 值表示。KIBA 数据集则融合了多个生物实验指标，用于衡量药物与靶点之间的综合相互作用强度。Human 与 C.elegans 数据集属于二分类任务，用于预测药物与靶点之间是否存在相互作用关系。

所有实验均使用五个不同随机种子重复实验。最终结果报告为所有实验结果的 mean $\pm$ std。

### 评价指标

对于回归任务，采用均方误差（MSE）、一致性指数（CI）以及 $rm^2$ 作为评价指标。其中，较低的 MSE 表示预测误差更小，而较高的 CI 与 $rm^2$ 表示模型具有更好的排序能力与回归拟合性能。

对于分类任务，采用 AUROC、AUPRC、Precision 与 Recall 作为评价指标。其中，AUROC 与 AUPRC 用于综合评估模型在不平衡数据集上的分类能力。

### 实现细节

本文实验环境配置如下：

- PyTorch 2.0.0 + CUDA 11.8
- PyTorch Geometric 2.6.1
- NVIDIA GPU 服务器（AutoDL 平台）

优化器采用 Adam，初始学习率设为 $1 \times 10^{-4}$。所有模型均训练至验证集性能收敛，并基于 validation set 指标选择最佳模型。

对于药物分子图编码部分，本文分别采用 DenseGINConv、DenseGCNConv、DenseGATConv 以及 DenseSAGEConv 作为图卷积算子，以验证不同图神经网络结构对模型性能的影响。

蛋白质序列统一截断或填充至长度 1000，并输入 TextCNN 模块进行特征提取。层次化池化模块中的默认超参数设置为：

- $n_1 = 6$
- $n_2 = 3$
- $\beta = 0.05$

其中，$n_1$ 与 $n_2$ 分别表示第一层与第二层图池化后的节点数量，$\beta$ 表示 MMD loss 的权重系数。

## 4.2 Overall performance comparison

Table~\ref{tab:davis_kiba_results} 与 Table~\ref{tab:human_celegans_results} 分别展示了 HAG-DTA 在回归任务与分类任务上的整体实验结果。

从实验结果可以看出，所提出的 HAG-DTA 在大多数评价指标上均优于现有基线模型，表明本文方法能够有效提升药物-靶标相互作用预测性能。

在 Davis 与 KIBA 数据集上，HAG-DTA 在 MSE 指标上取得了更低误差，同时在 CI 与 $rm^2$ 指标上获得了更优结果。这表明本文模型不仅能够降低回归预测误差，同时具有更好的排序一致性与整体拟合能力。

对于 Human 与 C.elegans 分类任务，HAG-DTA 在 AUROC 与 AUPRC 上均表现出较强性能，说明模型能够有效学习药物与蛋白质之间的潜在交互关系。此外，较小的标准差表明模型在不同随机初始化与数据划分条件下具有较好的稳定性与鲁棒性。

进一步分析发现，不同图卷积算子对模型性能存在一定影响。其中，GIN 在多数任务中表现最优，说明其较强的图结构判别能力更适合药物分子图建模任务；GCN 与 GraphSAGE 表现相对稳定，而 GAT 虽能够动态学习邻居重要性，但在部分数据集上存在一定性能波动。

总体而言，HAG-DTA 的性能提升主要来源于以下三个方面：

1. 层次化分子图表示能够有效学习药物分子的多尺度结构信息；
2. 深层图结构分支能够增强全局拓扑关系建模能力；
3. 基于注意力的多模态融合机制能够有效整合药物图与蛋白序列特征。

## 4.3 Comparison with recent state-of-the-art methods

为进一步验证本文方法的先进性，本文将 HAG-DTA 与近年来提出的多个 DTA 预测模型进行了比较，包括 GEFormerDTA、GNPDTA、WPGraphDTA 以及 MEGDTA 等近期方法。

实验结果如 Table~\ref{tab:sota_comparison} 所示。可以观察到，HAG-DTA 在多个数据集与评价指标上均取得了具有竞争力的结果。

与传统单尺度图表示方法相比，HAG-DTA 能够通过层次化图池化机制显式建模原子级、功能团级与分子片段级结构信息，从而提升药物结构表示能力。同时，本文所引入的全局图分支能够补充传统层次池化结构中可能丢失的全局拓扑信息。

此外，相较于部分仅依赖药物图结构的模型，HAG-DTA 通过 TextCNN 与注意力融合机制进一步增强了药物-蛋白多模态特征交互能力，因此在多个数据集上取得了更优性能。

## 4.4 Ablation studies

为验证 HAG-DTA 各模块的有效性，本文设计了一系列消融实验。实验结果如 Table~\ref{tab:ablation} 所示。

### 去除 MMD loss

首先，移除 MMD loss（即 $\beta=0$）后，模型性能出现明显下降。这表明 MMD loss 能够有效缓解局部层次特征与全局图特征之间的分布差异，从而提升特征融合效果。

### 去除全局图分支

当移除 DGNN 全局图分支，仅保留层次化池化结构时，模型性能同样下降。这说明仅依赖局部层次结构难以充分捕获药物分子的全局拓扑关系，而全局图表示能够提供重要补充信息。

### 去除注意力融合模块

进一步地，将注意力融合替换为简单特征拼接后，模型性能出现下降。这表明注意力机制能够动态学习不同特征分支的重要性，从而提升多模态特征融合能力。

### 去除蛋白质序列分支

当移除 TextCNN 蛋白质编码分支后，模型性能下降更加明显。这说明蛋白质序列特征对于药物-靶标相互作用预测具有重要作用，仅依赖药物图结构难以获得充分预测能力。

### 局部分支与全局分支对比

此外，本文进一步比较了仅使用局部分支（HGNN）与仅使用全局分支（DGNN）的情况。实验结果表明，两类结构具有明显互补性，其联合使用能够获得最佳性能。

## 4.5 Hyperparameter sensitivity analysis

为分析层次化池化结构中超参数设置对模型性能的影响，本文进一步研究了 $n_1$、$n_2$ 以及 MMD 权重系数 $\beta$ 的敏感性。

### n1/n2 参数分析

本文在：

- $n_1 \in {4,5,6,7,8}$
- $n_2 \in {2,3,4}$

范围内进行了网格搜索实验。

实验结果如 Figure~\ref{fig:n1n2_heatmap} 所示。

结果表明，不同数据集的最优 $(n_1,n_2)$ 组合存在差异。Davis 数据集在 $(n_1,n_2)=(4,2)$ 时获得最低 MSE 和最高 CI；KIBA 数据集倾向于稍大的第一层图结构，$(n_1,n_2)=(6,2)$ 表现最优，这与其较大的化合物集合规模一致；Human 和 C.elegans 数据集则在 $(n_1,n_2)=(7,3)$ 时取得最高 AUROC。

从模型角度分析，较小的 $n_1$ 会导致过度压缩，从而损失重要局部结构信息；而过大的 $n_1$ 则可能引入冗余簇结构与噪声信息。类似地，较小的 $n_2$ 难以充分学习高层次分子片段结构，而过大的 $n_2$ 则会降低层次池化的抽象能力。不同数据集的最优池化粒度存在差异，说明合适的层次压缩比与数据集的分子结构特性相关。

### MMD 权重分析

进一步地，本文研究了 MMD loss 权重 $\beta$ 对模型性能的影响。实验中：

[
\beta \in {0,0.01,0.05,0.1,0.5,1.0}
]

实验结果如 Figure~\ref{fig:mmd_beta} 所示。

结果表明，当 $\beta=0.05$ 时模型性能最佳。较小的 $\beta$ 无法有效约束不同特征分支之间的分布差异，而较大的 $\beta$ 则会过度强调分布对齐，进而影响主任务学习能力。

因此，$\beta=0.05$ 能够在预测性能与特征分布对齐之间取得较好平衡。

## 4.6 Complexity analysis

为分析 HAG-DTA 的计算复杂度，本文进一步统计了模型参数量、训练时间以及推理开销，结果如 Table~\ref{tab:complexity} 所示。

由于 HAG-DTA 引入了层次化图池化模块与多模态注意力融合结构，其参数规模与训练时间相比部分基线模型略有增加。然而，相较于模型性能提升幅度，其额外计算开销仍处于可接受范围内。

此外，实验结果表明，不同图卷积算子对模型复杂度同样存在影响。其中，GAT 由于需要显式计算邻域注意力，因此具有较高计算开销；GIN 与 GCN 在复杂度与性能之间具有较好平衡。

总体而言，HAG-DTA 能够在合理计算成本下获得较优预测性能。

## 4.7 Visualization and interpretability analysis

为进一步分析 HAG-DTA 的可解释性，本文对层次化图池化结果与注意力权重进行了可视化分析。

### 层次化池化可视化

Figure~\ref{fig:pooling_visualization} 展示了层次化池化过程中药物分子结构的变化。

第一层池化主要学习原子到功能团的聚合关系，而第二层池化进一步学习功能团到高层次分子片段的结构表示。

实验结果表明，模型能够自动学习具有化学意义的局部结构区域，而无需人工定义功能团规则。

### 注意力权重分析

进一步地，本文可视化了注意力融合模块中的权重分布情况。

结果表明，模型能够对部分关键原子区域与蛋白质局部序列赋予更高权重。这说明 HAG-DTA 能够自动关注潜在重要的结合相关区域。

### 药化学解释

从药化学角度来看，部分高权重区域对应于芳香环、含氮杂环以及可能参与氢键作用的功能基团。这些结果为识别潜在关键结合区域提供了有价值线索。

需要指出的是，本文可视化结果主要用于提供模型决策过程的初步解释，其所识别的重要子结构仍需进一步结合实验验证与分子对接分析进行确认。
